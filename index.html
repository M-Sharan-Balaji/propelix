Propelix helps HDB owners input property details once, pulls comparable market data, calculates a suggested best price, and then uses AI agents to publish the listing on 99.co.

Workflow: Profile → Analysis → Price Suggestion → Publish

What it does

  • Scrapes recent comparable HDB resale transactions via Tinyfish.
  • Calculates a suggested seller price using a clear, auditable pricing rule.
  • Logs into 99.co and automates listing creation on the owner’s behalf.

Ideal user

HDB owners who want to avoid underpricing, reduce admin overhead, and move from intention to live listing with fewer manual steps.
